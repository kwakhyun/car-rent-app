## RentCar

### Description

- React Native와 Expo를 이용해 개발한 렌트카 예약 앱입니다.
- JSON Server를 이용해 API 서버를 구현했습니다.
<br>

### Features

- 렌트카 상세 정보 확인
- 캘린더를 통한 예약 날짜 선택
- 렌트 비용 계산 및 예약
- 내 예약 내역 확인

### How to run

```bash
# 프로젝트 클론
$ git clone https://github.com/kwakhyun/rent-car.git

# 의존성 패키지 설치
$ yarn

# API 서버 실행
$ yarn api

# 프로젝트 실행
$ npx expo start
```
