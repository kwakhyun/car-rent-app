## Rent Car App

### Description

- React Native와 Expo를 이용해 개발한 렌트카 예약 앱입니다.
<br>

### Features

- 렌트카 조회 및 예약
- 내 예약 내역 확인

### How to run

```bash
# 프로젝트 클론
$ git clone https://github.com/kwakhyun/car-rent-app.git

# 의존성 패키지 설치
$ yarn

# 프로젝트 실행
$ npx expo start
```
